# Verificación de Instalación de Node.js

Después de instalar Node.js, ejecuta estos comandos en PowerShell o Terminal:

```powershell
# Verificar versión de Node.js
node --version

# Verificar versión de npm
npm --version
```

Si ambos comandos muestran números de versión, la instalación fue exitosa.

## Si aún no funciona después de instalar:

1. **Cierra completamente Visual Studio Code**
2. **Abre una nueva ventana de PowerShell como Administrador**
3. **Ejecuta los comandos de verificación nuevamente**

Si sigue sin funcionar, puede ser necesario reiniciar la computadora para que Windows actualice las variables de entorno.
